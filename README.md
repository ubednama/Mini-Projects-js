# 🖥️ Mini Projects JavaScript Collection

A retro terminal-inspired collection of 17 interactive web applications and mini-projects built with vanilla HTML, CSS, and JavaScript. Features a nostalgic terminal aesthetic with modern functionality and responsive design.

## 🌐 Live Demo

**[View Live Demo](https://mini-projects-js-five.vercel.app/)** - Explore all projects in action!

## ✨ Features

- **💻 Retro Terminal Design** - Nostalgic command-line interface aesthetic
- **📱 Fully Responsive** - Optimized for desktop, tablet, and mobile
- **⚡ Smooth Animations** - Terminal-style effects and interactions
- **🎨 Consistent Theming** - Universal CSS variables across all projects
- **🔧 Vanilla JavaScript** - No frameworks, pure web technologies
- **🎯 Accessibility First** - Keyboard navigation and screen reader support

## 🚀 Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Running Locally

```bash
# Clone the repository
git clone https://github.com/ubednama/Mini-Projects-js.git
cd Mini-Projects-js

# Open index.html in your browser
# Or use a local server:

# Python
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.
